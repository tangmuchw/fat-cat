熊熊卡主题提示组件

基于熊熊卡主题风格设计的全屏Loading与Toast提示组件
提供流畅的动画效果和友好的用户体验

全屏Loading

覆盖整个屏幕的加载提示，适用于需要长时间等待的操作

显示Loading
点击展示全屏加载效果
带文本Loading
显示自定义加载文本
隐藏Loading
关闭加载提示
JavaScript 控制方法
// 显示全屏Loading
showFullscreenLoading('正在加载中...');

// 隐藏全屏Loading
hideFullscreenLoading();

全屏Toast

覆盖整个屏幕的提示信息，支持多种状态类型

成功提示
操作成功时使用
警告提示
需要用户注意时使用
错误提示
操作失败时使用
信息提示
一般信息通知
JavaScript 控制方法
// 显示全屏Toast
showFullscreenToast({
    type: 'success', // 'success', 'warning', 'error', 'info'
    title: '操作成功',
    message: '您的卡片已成功发送！',
    duration: 3000 // 自动关闭时间(毫秒)，0表示不自动关闭
});

纯文字全屏Toast 新功能

简洁的文字提示，无需用户操作，自动关闭

纯文字提示
简洁消息提示，无需用户操作
加载提示
显示全屏加载效果
关闭所有
隐藏所有提示
JavaScript 使用方法
// 显示纯文字全屏提示
showTextToast('操作成功！', {
    duration: 2000, // 显示时长(毫秒)
    background: 'rgba(0, 0, 0, 0.8)', // 背景颜色
    textColor: 'white' // 文字颜色
});

// 显示带状态颜色的提示
showTextToast('存储空间不足，请及时清理', {
    type: 'warning' // success, warning, error, info
});

// 关闭所有提示
hideAllToasts();

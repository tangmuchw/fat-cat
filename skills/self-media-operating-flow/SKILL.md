# 自媒体从0到1：完整运营流程

## 阶段一：定位（第1–7天）

> 确定你是谁、为谁、提供什么价值。

1. **选领域**：列出自己有积累的1-3个方向，用微信/知乎/小红书搜关键词，评估竞争度。选一个主攻领域。

2. **定人群**：用公式“为【具体人群】解决【具体场景下的问题】”写出目标受众。忌“所有人”。

3. **找差异化**：分析同领域10个账号，从身份标签、内容形式、风格价值观中选出至少1个差异点。

4. **写定位声明**：150字简介，包含“我是谁、什么标签、分享什么、帮谁解决什么问题”。

## 阶段二：内容生产（持续）

> 建立选题→素材→写作→自检流水线。

5. **选题库**：每周储备至少30个选题，分类“常青/热点”。每天使用 `topic-selection` skill 筛选。

6. **素材库**：用Notion/飞书收集信源（知乎高赞、爆文、报告、书籍金句、个人经历），按选题标签存放。

7. **写作SOP**（参考 `high-value-writing` skill）：
    - 先定标题（含数字/强名词）
    - 搭框架（形式骨架、内容内核、传播引擎）
    - 填内容：每500字至少1个知识点
    - 加钩子：开头30字、中段共情反问、结尾具体征集
    - 优化可读性：段落≤5行、加粗、视觉锚点

8. **发布前自检清单**：
    - [ ] 标题0.3秒扫读有“与我有关”感？
    - [ ] 开头30字内共情？
    - [ ] 至少3处加粗？
    - [ ] 每500字有知识点？
    - [ ] 有个人IP露出？
    - [ ] 结尾有具体评论钩子？
    - [ ] 无流水账/说教/标题党？

## 阶段三：发布与运营（每篇）

> 分平台规则发布，冷启动。

9. **平台节奏**：
    - 公众号：每周2–3篇，固定时间
    - 知乎：每日回答1问 + 每周1篇长文
    - 头条：每日1篇，早7–9点或晚6–8点
    - 小红书：每周3–5篇，晚8–10点
    - 微博：每日3–5条短博 + 每周1篇头条文章

10. **发布动作**：
    - 按平台调标题长度（小红≤15字，头条可长）
    - 制作适配封面（公众号900×383，小红书3:4，头条16:9）
    - 加精准标签/话题（小红书3–5个，头条2–3个）
    - 文末直接写“评论区聊聊**\_\_**”
    - 多平台同步（微调开头标题）

11. **冷启动（前1000粉）**：
    - 公众号：转发朋友圈+社群，求点“在看”
    - 知乎：答高浏览量低回答数问题，文末引流
    - 小红书：参与官方话题，可投薯条测试
    - 头条：参与创作中心征文

## 阶段四：数据分析与迭代（每周）

> 不看数据等于盲打。

12. **核心指标**：
    - 阅读量低 → 改标题/封面
    - 完读率<30% → 优化开头和结构
    - 收藏高点赞低 → 增加情绪钩子
    - 评论低 → 检查评论钩子是否具体
    - 转发低 → 增加社交货币/嘴替时刻
    - 阅读高涨粉低 → 增加人设露出

13. **每周复盘**：
    - 记录本周发布篇数、爆款篇数
    - 分析最好和最差的一篇（标题、指标、原因）
    - 列出下周调整项（选题/标题/节奏/平台侧重）

## 阶段五：变现（粉丝1000+后启动）

> 分阶段接入收入。

14. **变现路径门槛**：
    - 0–1000粉：纯内容，不接广告
    - 1000–5000粉：可接置换或低价广告（100–500元），测试低单价产品（9.9–29.9元资料包）
    - 5000粉+：做付费咨询/社群/课程，广告报价 = 粉丝数×0.1–0.3元/阅读量

15. **常见变现方式**：
    - 平台流量收益（头条、公众号流量主）
    - 广告合作（粉丝1000+，小红书500+）
    - 知识付费/咨询（粉丝3000+且IP强）
    - 带货分销（粉丝500+）
    - 付费社群（粉丝2000+铁粉）

---

## 使用建议

- **时间线预期**：第1月起号，第2–3个月1000粉，第4–6个月2000–5000粉，第6个月后逐步变现。
- **必备工具**：选题（新榜/蝉小红）、素材（Notion）、写作（石墨/腾讯文档）、排版（Canva/黄油相机）、数据（各平台后台）。
- **核心心法**：前3个月只做内容和数据，半年后自然有收入。

# 缩写速查

| 英文缩写 | 释义                                          |
| -------- | --------------------------------------------- |
| VP       | Vice President，副总统、副总裁、负总监等职位  |
| VC       | Venture Capital，风险投资                     |
| BD       | Business Development，业务拓展                |
| ISV      | Independent Software Vendors，独立软件开发商  |
| PaaS     | Platform as a Service，是指平台即服务         |
| IaaS     | Infrastructure as a Service，即基础设施即服务 |
| -        | -                                             |

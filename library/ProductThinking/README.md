## 利用场景判断用户的真实需求

理查德·塞勒（2017 年诺贝尔经济学奖得主）早在 1980 年就提出的心理账户，如今已经是经济学中的重要概念。心理账户指的是，哪怕都是同样面值的收入和支出，在一个人的心中，它们都被分门别类做了编码，重要程度和账户的额度截然不同。

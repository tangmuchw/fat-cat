# 「慢拆·职业底盘」完整执行方案

## 一、系列定位

| 维度             | 内容                                                       |
| :--------------- | :--------------------------------------------------------- |
| **系列名称**     | 慢拆·职业底盘                                              |
| **Slogan**       | 用10期，把你的职业从“混沌”拆成“清晰”。                     |
| **核心命题**     | 我怎么知道自己适合做什么？怎么从“干得痛苦”变成“干得顺畅”？ |
| **系列总时长**   | 10期（图文/播客/视频均可）                                 |
| **更新节奏建议** | 每周1期，10周拆完一季                                      |
| **目标用户**     | 工作3-10年，正处于“上不去、下不来、不敢转”的职场卡顿期人群 |

> **为什么这个系列值得做？**
> 因为市面上的职业规划课都在教“怎么写简历”“怎么面试”——那是最后一公里。而用户真正卡住的，是前面的九十九公里：**不知道自己想要什么、不知道自己的优势在哪、不知道要不要转行、知道该动但就是动不了。**
>
> 这些，恰好是“拆³代”最擅长解决的问题：拆信息（看清职场真相）、拆框架（建立职业认知结构）、拆惯性（照见自己的逃避和假设）。

## 二、10期完整内容地图

整个系列遵循 **“拆信息→拆框架→拆惯性”** 的递进逻辑，但在10期的节奏上做了“轻重分配”：**前期重信息+框架（打地基）**，**后期重惯性（动手术）**，确保用户既能学到东西，又被逐步推向最深的自我照见。

### 【第一模块】拆信息·看清棋盘（第1-3期）

> **目标**：帮用户剥离外界噪音，看清职场的真实规则和自己此刻的真实位置。

#### 第1期：「拆来源」——“热门行业”是真的风口，还是有人在割韭菜？

| 拆解维度     | 内容要点                                                                                      |
| :----------- | :-------------------------------------------------------------------------------------------- |
| 拆来源       | 谁在说“这个行业好”？是猎头？是培训机构？是媒体软文？ta的立场是什么？                          |
| 拆事实与观点 | “AI人才缺口500万”是事实——但这个缺口要的是什么样的人？要几年经验？要什么学历？跟你有什么关系？ |
| 拆完整性     | 有人告诉你“转行做产品经理年薪60w”，但没告诉你“这是大厂P7的价，而大多数产品经理月薪15-25k”     |
| 拆框架       | （本期轻框架）引入一个简单工具：**信源评级矩阵**（信源可信度 × 与自身的相关度）               |
| 拆惯性       | （本期轻惯性）你过去做职业选择时，信息来源是什么？父母说、同学说、还是看了两篇软文就冲了？    |

> **本期结尾金句**：“不是所有风口都叫机会，有些叫镰刀。”
> **拆己作业**：写下你最近一次职业决策的3个信息来源，给每个信源打分（可信度/立场倾向/完整度）。

#### 第2期：「拆事实与观点」——你的工作到底“好不好”？把情绪从事实里拎出来

| 拆解维度 | 内容要点                                                                                               |
| :------- | :----------------------------------------------------------------------------------------------------- |
| 拆事实   | 客观事实：我的薪资是多少、工作时长是多少、汇报关系是什么、我负责什么范围——不带形容词地列出来           |
| 拆观点   | “我的工作好烂”——是你的判断。哪些具体事实支撑“烂”这个结论？列出来。                                     |
| 拆完整性 | 你只列了“烂”的证据，那“好”的证据呢？故意忽略的那一半长什么样？                                         |
| 拆框架   | （本期轻框架）引入 **“工作事实清单”模板**：分四栏【客观事实】【我的感受】【别人的评价】【客观数据】    |
| 拆惯性   | 你从小被教育“要谦虚”，于是养成了“只看自己不行的地方”的习惯——这份事实清单，你写“好”的那一栏时卡住了吗？ |

> **本期结尾金句**：“你觉得自己烂，可能只是因为你的眼睛只装了显微镜，没装广角镜。”
> **拆己作业**：用四栏模板重新描述你当前的工作，只写事实，不写形容词。

#### 第3期：「拆立场与完整性」——招聘JD里没写的那一半

| 拆解维度 | 内容要点                                                                                                                         |
| :------- | :------------------------------------------------------------------------------------------------------------------------------- |
| 拆来源   | 这份招聘信息是谁写的？HR写的（按模板填）、用人部门写的（真实需求）、还是猎头编的？                                               |
| 拆立场   | 招聘方说“我们招合伙人”——立场是希望你觉得有前景愿意少拿钱；说“公司氛围好”——立场可能是薪资没竞争力，拿氛围补                       |
| 拆完整性 | JD里写了“要求”“职责”，但没写“这个岗位最大的坑是什么”“前任为什么走”“真实的KPI长什么样”                                            |
| 拆框架   | （本期轻框架）引入 **“JD解码器”** ：一份JD拆成三层——明面要求（写了）、隐形门槛（没写但实际要的）、劝退条款（没写但你进去会痛的） |
| 拆惯性   | 你投简历时是不是只看“我能不能投”，从来不看“这个岗位合不合适我”？你的默认假设是不是“有工作要我我就该感恩”？                       |

> **本期结尾金句**：“JD是招聘方写的剧本，但你不是去演戏的，你是去工作的。”
> **拆己作业**：找一份你心仪的JD，用“JD解码器”拆出隐形门槛和劝退条款。

### 【第二模块】拆框架·建立坐标系（第4-7期）

> **目标**：给用户一套完整的认知框架，让ta能把“混乱的职业感受”放进有序的结构里看清楚。

#### 第4期：「拆结构」——职业满意度的三块拼图（能力×意愿×价值）

| 拆解维度 | 内容要点                                                                                                                           |
| :------- | :--------------------------------------------------------------------------------------------------------------------------------- |
| 拆结构   | 职业满意度不是一条线，是三根柱子：**能力**（我能不能做好）、**意愿**（我想不想做）、**价值**（我得到的回报是否符合预期）           |
| 怎么拆   | 画出三根柱子的长短。大多数人的痛苦来自三根柱子失衡：能力强但意愿低（倦怠）、意愿强但能力弱（焦虑）、能力强意愿也强但价值低（委屈） |
| 拆因果   | 你以为“薪资低所以没动力”——但有没有可能反过来：因为你没动力所以结果差所以薪资低？谁是因谁是果？                                     |
| 拆主次   | 三根柱子里，哪个是你当下的“最短版”？先补哪一根，收益最大？                                                                         |
| 拆惯性   | （本期进入惯性层）你是不是总盯着“价值”这根柱子？（“给我钱我就有动力了”）——你的惯性是用外部回报来覆盖内部问题                       |

> **本期结尾金句**：“职业痛苦不是你不努力，是你的三根柱子长短不齐。”
> **拆己作业**：给“能力·意愿·价值”三栏分别打分（1-10分），并写下最短的那根柱子的具体表现。

#### 第5期：「拆结构」——能力拆解：你不是“不行”，你是“哪块不行”没拆开

| 拆解维度 | 内容要点                                                                                                           |
| :------- | :----------------------------------------------------------------------------------------------------------------- |
| 拆结构   | “能力”拆成三层：**专业技能**（你会什么）、**通用能力**（沟通/协调/解决问题）、**底层素质**（学习力/抗压力/责任感） |
| 怎么拆   | 你的“能力不行”是哪一层不行？很多人的问题是：专业技能80分，但通用能力30分，导致整体评价不及格                       |
| 拆因果   | 你觉得“公司不重视我”——有没有可能是因为你的能力只有自己知道，没有转化成被看见的表达？                               |
| 拆主次   | 当前阶段，哪一层能力的提升回报率最高？不是补短板，是“在优势区加杠杆”                                               |
| 拆惯性   | 你默认“有能力就够了，不用展示”——这个假设从哪来的？是不是“酒香不怕巷子深”这句老话把你坑了？                         |

> **本期结尾金句**：“能力是里子，但别人只能看见面子。你不拆开，别人看不见。”
> **拆己作业**：用三层模型拆你的能力结构，标出最高的一项和最低的一项。

#### 第6期：「拆因果」——你的“职业倦怠”，到底是因还是果？

| 拆解维度 | 内容要点                                                                                           |
| :------- | :------------------------------------------------------------------------------------------------- |
| 拆因果   | 你以为是“工作太累→所以我倦怠”。但有没有可能反过来：“我倦怠→所以觉得工作特别累”？                   |
| 拆结构   | 倦怠的两个来源：**消耗型倦怠**（干太多）vs **空虚型倦怠**（干的事没意义）。你属于哪一类？          |
| 拆主次   | 如果是消耗型→需要的是边界管理；如果是空虚型→需要的是意义重构。你错配了解决方案吗？                 |
| 拆步骤   | 恢复动力的顺序：先解决身体层面的耗竭，再解决认知层面的困惑。你跳过第一步直接想“我该不该辞职”了吗？ |
| 拆惯性   | 你的惯性是不是“一累就想逃”（换工作/辞职/躺平），而不是先拆开看“是哪一种累”？                       |

> **本期结尾金句**：“倦怠是个垃圾筐，所有不想面对的问题都被扔进去了。”
> **拆己作业**：写下你最近的“累”，用两类倦怠做区分，并标注比例（消耗型占多少%/空虚型占多少%）。

#### 第7期：「拆步骤」——转行/跳槽的正确顺序，以及你为什么总是跳反

| 拆解维度 | 内容要点                                                                                                         |
| :------- | :--------------------------------------------------------------------------------------------------------------- |
| 拆步骤   | 正确顺序：**定位**（我要去哪）→ **盘点**（我有什么）→ **补缺**（还差什么）→ **行动**（开始投）。你跳过哪一步了？ |
| 拆因果   | 很多人“先辞职再想”——这个顺序带来的后果是什么？（仓促选择、压低身价、焦虑驱动决策）                               |
| 拆结构   | 跳槽的两个维度：**行业**（换不换）和 **职能**（变不变）。四象限里，哪个象限风险最低、哪个最高？                  |
| 拆主次   | 跳槽中最关键的不是“新工作薪资涨了多少”，而是“旧工作的核心问题在新工作中是否被解决”                               |
| 拆惯性   | 你是不是用“换环境”来逃避“面对自己”？换一个公司，不换思维，不过是换个地方继续卡。                                 |

> **本期结尾金句**：“跳槽解决的是位置问题，解决不了你的惯性问题。”
> **拆己作业**：如果你正在考虑跳槽，写下“我想逃离的3件事”和“我想追求的3件事”，并标注哪些是外部可变的、哪些是内部必带的。

### 【第三模块】拆惯性·照见自己（第8-10期）

> **目标**：这是全系列的灵魂——帮用户看见困住自己的那条“思维老路”。

#### 第8期：「拆默认假设」——“我不够好/我没资格”到底是谁告诉你的？

| 拆解维度   | 内容要点                                                                                             |
| :--------- | :--------------------------------------------------------------------------------------------------- |
| 拆默认假设 | 你脑子里有没有这些声音：“我学历一般所以没资格要高价”“我经验不够所以不能提要求”“我不如别人”           |
| 怎么拆     | 这个“不够好”的判断，有没有证据？证据是什么时候的？现在的你还和那时一样吗？                           |
| 拆情绪逻辑 | “我不配”是一种感觉，不是事实。但这个感觉驱动了你多少职业行为？（不敢谈薪、不敢争取、总是选次优选项） |
| 拆结构     | 用“证据天平法”：左边列“我不够好”的证据，右边列“我做得不错”的证据。天平真的偏向左边吗？               |
| 拆思维定式 | 你是不是有一个固定模式：**成功归外部（运气好），失败归内部（我太差）**？                             |

> **本期结尾金句**：“‘我不够好’是你心里住着的一个老评委，ta的评分标准已经过期了。”
> **拆己作业**：完成“证据天平法”，看看天平的真实倾斜方向。

#### 第9期：「拆逃避模式」——你嘴上说“再想想”，其实是“不敢选”

| 拆解维度   | 内容要点                                                                                    |
| :--------- | :------------------------------------------------------------------------------------------ |
| 拆逃避模式 | “我再想想”“我再准备准备”“等一个更好的时机”——这些话说多少遍了？                              |
| 怎么拆     | 逃避的两个核心动因：**害怕选错**（完美主义）vs **害怕承担**（不想为结果负责）。你是哪一种？ |
| 拆因果     | 你以为“想清楚才能行动”——但有没有可能，不行动永远想不清楚？行动本身就是想的一部分？          |
| 拆步骤     | 正确顺序：**小步试探→获取反馈→调整方向**。你的错误顺序：**无限思考→更焦虑→更不敢动**        |
| 拆情绪逻辑 | “再想想”带来的虚假掌控感，让你误以为自己在推进，其实只是在原地兜圈                          |

> **本期结尾金句**：“想和做之间隔着的不是准备，是恐惧。”
> **拆己作业**：写下你“想了超过3个月但还没行动”的一件事情，标出你真正怕的是什么。

#### 第10期：「系列收官」——拆出你自己的“职业使用说明书”

| 拆解维度   | 内容要点                                                                                                                                                |
| :--------- | :------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 本期任务   | 带用户回顾10期拆解成果，整合成一份**个人专属的职业使用说明书**                                                                                          |
| 说明书结构 | ①我的能力结构图（来自第5期）②我的三根柱子现状（第4期）③我的信息来源清单（第1期）④我的逃避模式画像（第9期）⑤我的默认假设清单（第8期）⑥我的下一步行动方案 |
| 拆惯性     | 最后一层拆：**你是不是一直在等一个“确定的答案”** ——但职业没有确定答案，只有“当前最优”和“下一步试探”                                                     |
| 系列终局   | 你不是被“治愈”了，你是被“看见”了。看见了路，看见了车，看见了自己在哪——剩下的就是踩油门。                                                                |

> **本期结尾金句**：“职业底盘不是找到的，是拆出来的。”
> **拆己作业**：完成完整的“职业使用说明书”，并把“下一步行动”具体到一件明天就能做的事。

## 四、内容创作规范（每期固定格式）

| 模块           | 要求                                                                                                                       |
| :------------- | :------------------------------------------------------------------------------------------------------------------------- |
| **标题公式**   | 具体场景 + 三层拆完后的结论反差。例：“为什么你每天加班却没有成就感？——不是你不努力，是你的三根柱子歪了。”                  |
| **开头**       | 扔出一个具体场景/用户的真实留言/一个共情力极强的问题，让用户觉得“这说的就是我”。                                           |
| **正文结构**   | 严格按“拆信息→拆框架→拆惯性”推进，每层用小标题标注，让用户有清晰的“进度感”。                                               |
| **每期必含**   | ①核心工具/模型 ②一个“拆己作业”（用户必做） ③结尾金句                                                                       |
| **评论区运营** | 每期置顶一条“交作业区”，鼓励用户晒拆己作业，作者亲自回复前10个认真交作业的人——这既是建立深度连接，也是训练营的潜在生源池。 |
| **视觉统一**   | 每期固定封面风格、固定“进度条”位置（如：慢拆·职业底盘 ③/⑩），制造系列感和追更欲。                                          |

**这份方案的核心打法是：用10期免费内容建立专业信任 + 用每期的“拆己作业”筛选有行动力的用户 + 用工具包完成首次付费 + 用训练营和私教承接深度需求。**

整个系列做下来，“拆³代”不仅是一个内容账号，更是用户在职业这件事上“信得过的手术台”。你先从第1期的内容写起，还是先做「JD解码器」工具包？

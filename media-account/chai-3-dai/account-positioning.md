# 账号定位方案：「拆³代」（完整版）

## 一、一句话定位

**拆³代 = 不是创作，是总结。不是道理，是工具。不是教人思考，是陪你走完。**

## 二、核心价值

> 我不是在创作，是在总结。
> 不是给道理，是给工具。
> 不是教人思考，是陪你走完。

这三句话构成完整闭环：

| 层级   | 核心价值                 | 对应三层拆法 | 解释                                 |
| :----- | :----------------------- | :----------- | :----------------------------------- |
| 第一句 | 我不是在创作，是在总结   | 拆信息       | 信息筛选与还原，从海量信息中捞出真相 |
| 第二句 | 不是给道理，是给工具     | 拆框架       | 结构重组与路径梳理，把方法封装成工具 |
| 第三句 | 不是教人思考，是陪你走完 | 拆惯性       | 照见自己，走过从"知道"到"做到"的全程 |

**三层递进关系：先看清事实 → 再建立结构 → 最后照见自己。**

**品牌记忆（一句话让人记住你）：**

> **拆³代——把思维装进工具里**

学不会顶级思维，就用顶级工具。这就是拆³代存在的唯一目的。

## 三、你在做什么

你走的不是"内容创作"这条路，是"总结"这条路。

你走了很长的路，回头看，把当年缺的东西找出来，把后来长出来的那套思考方式和方法，总结成别人能直接拿走用的东西，放在这里。

你的内容全部来自于**"总结"** ——把过去的经验、踩过的坑、长出来的方法，封装成一个个工具。

**你不是"教人思考"，你是把顶级思维拆了、装成工具，让人拿来就用。**

思维固化的人，直接学顶级思维——培养周期太长、门槛太高、现实不允许。等思维养成了，当下的问题早把人压垮了。

所以你不教思维，你教工具。因为工具的本质——就是"顶级思维被封装好了的样子"。读者不用成为顶级高手，只需要拿上你给的工具，劈自己当下的柴。

**你不是教人思考，你是帮人"做思维的内部大扫除"。**

别人拆解世界是为了让用户"变聪明"，你拆解世界是为了让用户"放过自己，然后重新选路"。

## 四、你解决什么问题

市面上所有内容平台的"答案"都有一个共同缺陷：

**帮你"绕过"问题，而不是"穿过"问题。**

| 平台   | 问题                                                 |
| :----- | :--------------------------------------------------- |
| 知乎   | 给结论，不给路径——让你觉得"我知道了"，然后停下来     |
| 小红书 | 给指南，不给诊断——告诉你怎么做，不告诉你为什么做不到 |
| 短视频 | 给情绪，不给工具——让你被理解，关上手机还是老样子     |

**你的解法完全不同：**

| 维度 | 其他平台              | 拆³代                      |
| :--- | :-------------------- | :------------------------- |
| 起点 | 告诉你"你应该"        | 指认你正在经历什么         |
| 过程 | 给结论（1、2、3步走） | 一层层走完，每一步都有工具 |
| 终点 | 让你觉得"我知道了"    | 让你带走一个能用的东西     |
| 姿态 | 老师/专家/嘴替        | 走过同样路的人             |

## 五、内容框架：三层递进

> 每一期内容都走完这三层，才算"陪你走完"。

| 层级   | 名称   | 本质     | 核心动作                             | 用户收获                   |
| :----- | :----- | :------- | :----------------------------------- | :------------------------- |
| 第一层 | 拆信息 | 客观还原 | 筛信源、分事实与观点、揭露立场       | "我不再被带着走了"         |

**三层合一，才算拆透。**

## 十三、执行建议

1. **品牌记忆已经定下来了。** "拆³代——把思维装进工具里。" 不解释、不展开，放在简介里，让读者自己完成"这人手里有东西"的判断。

2. **按平台裁剪长度，不套同一套话。** B站知乎用完整版，小红书去第一句，抖音取最后两句。

3. **内容按公式走。** 每期都是一次完整的"陪你走完"——指认困境→给工具→带走动作。

4. **始终站在"总结者"的位置上说话。** 不是"我在教你"，是"我总结好了，放在这里，你拿走用"。

5. **第三层（拆惯性）是核心差异化。** 市面上的内容很少走到这一层。走到这里，用户回不去。

6. **每一篇文章写完后，问自己一个问题：** 这篇文章把什么思维，装进了什么工具里？如果答不上来，重写。

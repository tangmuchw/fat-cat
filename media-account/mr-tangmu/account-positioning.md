## 二、价值定位（含差异化依据）

### 1. 核心价值主张

> **“用人文的视角解构职场，给内向者一套‘体面生存’的心法与技法。”**

- **表层价值**：可落地的工具（如天平模型、三级话术）——敲门砖
- **深层价值（护城河）** ：情绪脱敏与自我重建——“走过一遍，就更懂想要什么”

### 3. 差异化判断四大依据

#### 依据二：竞品红海缺口分析

| 红海类型       | 代表风格          | 他们的短板                   | 你的差异化占位                 |
| -------------- | ----------------- | ---------------------------- | ------------------------------ |
| “狼性厚黑派”   | 讲权谋、教斗同事  | 缺乏对内向者、高敏感者的共情 | **填补“共情与解法”之间的鸿沟** |
| “冰冷的工具人” | AI写作、Excel模板 | 没有人格温度，用户黏性差     | **用温度包裹工具**             |
| “鸡汤安慰派”   | “你要爱自己”      | 只给汤不给勺，看完更焦虑     | **给情绪接纳，更给可执行模型** |

**结论**：你的定位是 **“有工具的温柔”** ——既不冰冷，也不空洞。

#### 依据三：创作者基因锁（你是谁）

- **名字基因（汤木）** ：来自汤木的励志文学→底子是**人文主义**
- **经历基因（走过一遍）** ：反思型、内倾型→对**内耗、自我认同、表达焦虑**有切肤之痛
- **形成结论**：内容必须带“自我袒露”和“思辨性”——教人“如何在打工中保全自我、找到意义”，这是别人模仿不来的护城河

#### 依据四：用户隐性需求洞察

- **嘴上说的需求**：“教我怎么写绩效自评”
- **心里真正的隐痛**：“我觉得自己没价值，写不出来是因为我不认可自己”
- **形成结论**：差异化不是“教得快”，而是 **“教得透”** ——把职场问题心理学化，区别于所有“模板号”

## 三、人格化定位

### 1. 人设角色

**“归来的学长”——一位刚走过泥泞路、手里拿着湿漉漉地图的同行者**

- 不居高临下教做人，而是“我懂你，因为我刚经历过”
- 不制造焦虑，而是“别怕，这里有方法”

### 2. 人格调性

| 场景         | 语调风格                                                           |
| ------------ | ------------------------------------------------------------------ |
| 写职场干货时 | **理性的外科医生**——解剖问题，给工具模型（如证据天平、老族长理论） |
| 写心境感悟时 | **深夜陪你喝酒的老友**——不劝你忘掉，而是陪你“走过一遍”             |
| **关键特质** | **诚实**——有AIGC成分？坦然聊“AI时代的写作者如何自处”，增强信任感   |

### 3. 人格化三要素

- **可信赖**：不装、不吹、不教厚黑
- **可亲近**：有温度、有共情、不端着
- **可向往**：比读者先走过一步，但只领先半步——让读者觉得“我也能做到”

## 五、初始化定位（未来3个月执行方案）

### 1. 内容断舍离

| 动作        | 具体内容                                                                |
| ----------- | ----------------------------------------------------------------------- |
| ✅ **砍掉** | 纯情感自述（《失忆》类）——数据验证无效                                  |
| ✅ **合并** | 把“情感”作为“职场”的底色——所有职场痛点，最后落到“个人的成长与和解”      |
| ✅ **死磕** | “职场心理+工具模型”（《证据天平》《老族长》类）——数据验证过的“本命赛道” |

### 2. 三大内容底盘

#### 底盘一：职场成长池（最容易爆、最吸粉）

选题方向：写不出来（绩效/周报/述职）、说不出来（汇报/沟通/拒绝）、想不明白（辞职/转行/回老家）、扛不住了（加班/PUA/35岁危机）

选题示例：

- 《为什么你越努力工作，越得不到认可？》
- 《戒掉这3个习惯，我终于不再害怕跟领导说话》
- 《真正厉害的职场人，早就不“努力”了》

#### 底盘二：认知感悟池（最容易转粉、最适合做个人IP）

选题方向：走过一遍后的顿悟、自我和解、配得感重建

选题示例：

- 《走过一遍，就比原先更懂自己想要什么》——你的slogan本身就是一个系列
- 《我是什么时候开始不再害怕被否定的》
- 《写给那个在绩效自评前焦虑到失眠的你》

#### 底盘三：读书方法论池（最容易持续、最有长期价值）

选题方向：从汤木的书出发，结合职场场景做深度解读

选题示例：

- “一本书解决一个职场问题”系列

### 4. 爆款选题公式（直接套用）

| 公式          | 示例                                  |
| ------------- | ------------------------------------- |
| 痛点式        | 《为什么你越忙，越得不到认可？》      |
| 结果式        | 《戒掉这3个习惯，我终于不再害怕汇报》 |

- 为什么你越\_**\_，越\_\_**？
- 戒掉\_**\_，我终于\_\_**。
- 真正\_**\_的人，早就不\_\_**了。
- 写给正在\_\_\_\_的你。

### 5. 思维导图选题法

以**“职场人的痛苦”**为根节点，向下发散：

- **写不出来** → 绩效自评、周报、述职报告、年终总结
- **说不出来** → 跟领导汇报、跟同事沟通、拒绝别人、背锅防坑
- **想不明白** → 要不要辞职、要不要转行、要不要考研、要不要回老家
- **扛不住了** → 加班、内卷、PUA、35岁危机

每一个细分痛点 = 一篇潜在爆款。

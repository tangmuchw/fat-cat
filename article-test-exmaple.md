你有没有注意过自己描述工作时的用词？

“我的工作好烂。”
“我什么都做不好。”
“这份工作没什么意思。”

说的时候，你觉得这些是事实。但你停一下——“烂”是事实吗？

**事实是：方案被退回来改了三次。**
**你的判断是：我好差。**

这是两件完全不同的事。

你每一天都在做这个转换：**把具体事件，翻译成对自己的否定。**

方案被退回来 → “我不行。”
领导没回消息 → “他不重视我。”
同事做得好 → “我比不上他。”

翻译本身没有对错。问题在于：**你忘了你翻译过。**

你把翻译后的版本直接当成了事实，然后开始为这个“事实”焦虑。

我们来看看这两列东西：

| 事实                | 你的翻译版本     |
| :------------------ | :--------------- |
| 方案被退回修改了3次 | 我好差           |
| 每天工作9小时       | 这份工作太消耗了 |
| 入职两年没有升职    | 我没前途         |
| 领导没回应我的建议  | 他不重视我       |
| 月薪X元             | 我的价值只有X元  |

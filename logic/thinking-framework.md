# 人与人最核心的 5 种思维框架

---

# 1. 点状思维（最底层）

**特点：只看眼前、只看单点、没有因果、没有结构**

- 看问题：只看当下情绪、当下结果
- 思考：想到哪说到哪，容易被带节奏
- 行为：冲动、短视、情绪化
- 典型表现：
    - 只看眼前利益，不看长期代价
    - 别人说啥信啥，缺乏独立判断

**适合：简单执行、不需要复杂判断的事**
**局限：一遇到复杂问题就乱、就慌、就抱怨**

---

# 2. 线性思维（最常见）

**特点：单线因果、非黑即白、简单归因**

- 逻辑：A → B，只有一条线
- 喜欢：找“唯一原因”“唯一答案”
- 常见误区：
    - 努力 = 成功
    - 有钱 = 幸福
    - 对我好 = 好人

**优点：简单、高效、好理解**
**局限：世界是复杂系统，线性思维经常判断错误**

---

# 3. 结构思维（高手入门）

**特点：有框架、有分类、有层次、抓重点**

- 看问题：先拆结构，再看关系
- 做事：先搭框架，再填内容
- 常用工具：
    - 分类、归纳、优先级
    - 问题→原因→方案→落地
- 典型人群：产品、管理者、工程师、策划

**优点：逻辑清晰、表达清楚、做事靠谱**
**核心：把混乱变有序**

---

# 4. 系统思维（中高阶）

**特点：看整体、看循环、看关系、看长期**

- 不只看单点，看**要素 + 关系 + 反馈**
- 能理解：
    - 延迟效应（现在不代表未来）
    - 隐性影响（看不见的规则）
    - 恶性循环/良性循环
- 典型人群：创业者、决策者、资深管理者

**优点：能看透本质、不容易被表面迷惑**

---

# 5. 元思维（顶层）

**特点：思考“自己的思考”**

- 我为什么这么想？
- 我的偏见是什么？
- 我用的框架对不对？
- 我如何升级自己的思维？

这是**唯一能不断自我进化**的思维。

---

# 一句话总结差距

- 点状：**看不到联系**
- 线性：**只看到一条线**
- 结构：**能搭出骨架**
- 系统：**看懂整体运转**
- 元思维：**能升级自己**

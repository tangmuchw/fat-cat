# word-break & word-wrap

## word-break

| 值        | 描述                               | 备注                                                             |
| --------- | ---------------------------------- | ---------------------------------------------------------------- |
| normal    | 使用浏览器默认的换行规则（会溢出） | -                                                                |
| break-all | 允许在单词内换行                   | **会尝试把长单词挪到下一行，而是直接进行单词内的断句，节省空间** |
| keep-all  | 只能在半角空格或连字符处换行       | -                                                                |

## word-wrap

| 值         | 描述                               | 备注                                                                                                         |
| ---------- | ---------------------------------- | ------------------------------------------------------------------------------------------------------------ |
| normal     | 使用浏览器默认的换行规则（会溢出） | -                                                                                                            |
| break-word | 在长单词或 URL 地址内部进行换行。  | **首先尝试挪到下一行，看看下一行的宽度够不够，不够的话就进行单词内的断句，很容易出现一片一片空白区域的情况** |

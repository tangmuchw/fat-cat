# css 权重

| 类型                        | 权重 |
| --------------------------- | ---- |
| !important                  | 无穷 |
| 行内样式                    | 1000 |
| id                          | 100  |
| class/属性选择器/伪类:hover | 10   |
| 标签选择器/伪元素:after     | 1    |
| 通配符                      | 0    |

## 示例

```CSS
li:first-line {} /* 2 */

ul ol+li{} /* 3 */

h1+[rel=up]{} /* 11 */

div p.box{} /* 12 */

```
